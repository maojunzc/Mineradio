# Mineradio v1.1.0 纯净安装版

这是 Mineradio 1.1.0 的纯净安装发布版。安装包从当前可信源码重新构建。

## 主要更新

- 将 `默认测试.json` 设置为首次启动默认用户存档，软件内默认视觉参数也同步使用这份快照。
- 新用户首次启动时，用户存档列表会自动预置「默认测试」。
- 默认视觉参数更新为：`emily` 预设、封面粒子分辨率 `1.55`、电影镜头强度 `0.5`、歌词发光 `0.28`、高亮色 `#fac900`、发光色 `#008aff`、UI/Home/视觉图标白色。
- 默认 3D 歌单架为静态常驻侧栏，侧向角度 `-15`，播客歌单默认关闭。
- 3D 歌单架新增「显示播客歌单」和「合并收藏歌单」开关。
- 高级设置新增后台策略和画质档位，直播后台保持开启后不会进入低占用暂停。
- 修复用户存档应用后切歌/跳歌回退到上一个视觉预设的问题。
- 修复 3D 歌单详情页动态/静态镜头绑定混乱、常驻触发区异常、详情页遮挡和滚轮高亮不同步问题。
- 修复详情页选歌时歌词透明度跳亮；详情页打开时歌词保持可读，但不遮挡中心高亮行。
- 修复全屏 DIY 悬浮入口遮挡视觉预设热键按钮，以及视觉控制台隐藏按钮残留的问题。
- 更新 Electron / electron-builder 依赖，并保留更新下载失败时的安全停住逻辑。
